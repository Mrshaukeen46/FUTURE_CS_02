## üß© Incident Response Process
1. **Detection** ‚Äî Identify alerts and extract logs from SIEM dashboard.  
2. **Analysis** ‚Äî Verify anomalies, correlate IPs, and evaluate event patterns.  
3. **Classification** ‚Äî Assign severity (High/Medium/Low).  
4. **Response** ‚Äî Contain affected systems, block malicious IPs, patch vulnerabilities.  
5. **Recovery** ‚Äî Restore system operations and validate patch success.  
6. **Reporting** ‚Äî Prepare and communicate findings to management.
